# Visual Direction Matrix

Use this to avoid repeating the same AI-generated SaaS look.

## Scene metaphors

- Command room
- Field notebook
- Private desk
- Editorial archive
- Trading desk
- Studio console
- Operations cockpit
- Client evidence room
- Compliance ledger
- Calm account center

## Density levels

- Calm: wide spacing, few decisions per screen
- Standard: normal SaaS density
- Compact: efficient but readable
- Operator-grade: high information density with strong hierarchy
- Editorial: content-first, typographic hierarchy
- Immersive: visual-first but still task-safe

## Surface styles

- Paper-like surfaces
- Tinted dark surfaces
- Matte panels
- Thin-lined workspace
- Ledger-like rows
- Quiet cards
- Split-pane workbench
- Document-first canvas

## Proof styles

- Logs
- Timelines
- Source notes
- Screenshots
- Audit trails
- Comparisons
- Checklists
- Annotated records
- Before and after states

## Motion styles

- Instant feedback
- Subtle fade and translate
- State transition only
- Drawer reveal
- Row expansion
- Toast confirmation
- Reduced-motion equivalent
